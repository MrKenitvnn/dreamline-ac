Là link banner quảng cáo do Affiliate tự tạo kết nối với các sản phẩm hay các trang web sự kiện theo các link của Merchant cung cấp trên trang Adpia.

Phụ thuộc theo loại link mà có thể chỉ tạo ra các đường dẫn đã bị thay đổi link banner, link text hay Adpia code. Phụ thuộc theo tình trạng nhập hay không nhập Merchant ID, chia thành 2 loại lớn là loại 1 và loại 2.

Chọn loại

Loại 01 : Không cần nhập Merchant ID riêng, chỉ cần nhập Link Targer mình thích là có thể tạo được link của Merchant tương ứng.
Loại 02 : Đầu tiên phải nhập Merchant ID sau đó nhập LInk Target thì mới tạo được link.
